# Java 8 JRE

Installs Java8 JRE on a Debian-based Linux host.

## Requirements

Debian-based Linux.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

```
    - hosts: servers
      roles:
         - { role: 4ARMED.java8 }
```
